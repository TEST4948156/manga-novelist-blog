🌐 Netlify 拖拽部署（推荐）

最简单的部署方式，无需任何技术配置

  1. 点击下方按钮打开 Netlify
  2. 注册/登录账号
  3. 将 dist 文件夹拖拽到部署区域
  4. 等待部署完成，获得永久 URL
🚀 立即部署到 Netlify

📚 GitHub Pages

免费托管，与 GitHub 完美集成

  1. 将项目推送到 GitHub 仓库
  2. 在仓库设置中启用 GitHub Pages
  3. 选择 dist 文件夹作为源
  4. 获得 GitHub Pages URL
📚 前往 GitHub

💻 本地服务器（测试用）

在本地快速预览部署效果

Python 方式：
cd dist
python -m http.server 8080
Node.js 方式：
npx serve dist -p 8080
PHP 方式：
cd dist
php -S localhost:8080
